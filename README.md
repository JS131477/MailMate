# MailMate - 邮件智能助手

MailMate 是一款基于 AI 的 Outlook 插件，旨在提升办公效率，为员工提供邮件翻译、语气分析、智能回复建议和中英文转换等核心功能。

## 核心功能

1. **自动翻译**：将外文邮件实时翻译为用户首选语言，支持英文、中文、法语、德语、西班牙语等多种语言。

2. **语气分析**：识别发件人的情绪倾向（积极/中性/消极）、正式程度（正式/半正式/随意）和紧急程度（高/中/低）。

3. **智能回复建议**：基于邮件内容和语气分析结果，生成 2~3 条不同风格的回复草稿（正式、友好、简洁）。

4. **中文输入转英文**：当用户用中文输入回复内容时，自动将其转换为地道、专业的英文邮件正文。

## 技术栈

- **前端**：React + Office.js
- **构建工具**：Webpack
- **开发环境**：Node.js

## 安装与使用

### 开发环境设置

1. 克隆项目到本地：
   ```bash
   git clone <repository-url>
   cd mailmate
   ```

2. 安装依赖：
   ```bash
   npm install
   ```

3. 启动开发服务器：
   ```bash
   npm start
   ```

4. 在 Outlook 中加载插件：
   - 打开 Outlook
   - 点击 "文件" > "管理外接程序"
   - 点击 "添加" > "从文件添加..."
   - 选择项目根目录下的 `manifest.xml` 文件
   - 点击 "安装"

### 生产环境部署

1. 构建项目：
   ```bash
   npm run build
   ```

2. 将 `dist` 目录下的文件部署到您的服务器

3. 更新 `manifest.xml` 中的 URL 为您的服务器地址

4. 在 Outlook 中加载更新后的 manifest 文件

## 功能使用说明

### 邮件翻译

1. 打开一封邮件
2. 点击邮件工具栏中的 "翻译邮件" 按钮
3. 在任务窗格中查看翻译结果
4. 可选择 "显示原文/译文对照" 或 "仅显示译文"

### 语气分析

1. 打开一封邮件
2. 在任务窗格中点击 "分析语气" 按钮
3. 查看语气分析结果，包括情绪、正式程度和紧急程度

### 智能回复

1. 打开一封邮件
2. 点击邮件工具栏中的 "智能回复" 按钮
3. 在任务窗格中查看生成的回复建议
4. 选择一个回复风格并点击 "使用此回复" 按钮插入到邮件中

### 中文输入转英文

1. 开始撰写新邮件或回复邮件
2. 在任务窗格中输入中文内容
3. 点击 "转换为英文" 按钮
4. 查看转换结果并点击 "插入到邮件" 按钮

## 注意事项

- 本插件需要网络连接以使用 AI 服务
- 语气分析仅用于辅助理解，不存储个人情感数据
- 首次使用时可能需要授权访问邮件内容
